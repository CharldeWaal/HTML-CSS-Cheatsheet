Useful HTML Elements

Document Declaration and Starting Blocks

Tag Name Useful Attributes Description Example Code
<!DOCTYPE html> Document Type - Declare the document type as html <!DOCTYPE html>
<html> html tag - All content should be wrapped in a html tag to insure that web browsers interpret the html code contained in the document correctly. The html element, the root of an HTML document, should be added after the !DOCTYPE declaration. <html>...</html>
<head> head - Declares the head of the html document. The head should contain the web page title, links to external files and stylesheets and scripts. <head>...</head>
<title> title - Declares the web page's title. This title can be seen as the text in the webrowsers tab. <title> 'insert website title here' </title>
<link> link href="url or relative path" rel="stylesheet" type="text/css" The link tag is used to link external files, such as a CSS file. <link href="./styles.css" rel="stylesheet" type="text/css">

Document Structure (Semantic HTML)

Tag Name Useful Attributes Description Example Code
<header> header section - Semantic HTML for a websites header section. This replaces using a div element to create a header in previous versions of HTML. This tag is not to be confused with the <h1> header and <head> head tags, which serve different purposes entirely. <header>...</header>
<main> main section - Semantic HTML used to create a websites main body section. This replaces using a div element to create a section in previous versions of HTML. <main>...</main>
<footer> footer section - Semantic HTML used to create a web page's footer section. This replaces using a div element to create a footer section in previous versions of HTML. <footer>...</footer>
<section> section - Semantic HTML used to group multiple elements into a single section of a web page. <section>...</section>
<article> article - The article tag is a semantic HTML tag that is used to wrap text. This is done to ensure that web browsers and document readers create the correct document outline of our wep page. Wrapping content in a article tag also enables other webistes to easily reference certain sections of our web pages content by linking to the appropriate article. This also ensures better SEO results for our web page. The content between article tags should generally make sense on it's own. <article>...<article>
<aside> aside - Aside tags are semantic HTML tags that are used to include additional information that can enhance another element but is not required. They are used to include out of context information. <aside>...</aside>

<strong> Strong Element - The Strong element highlights important text, making it bold <strong>...</strong>
<span> Span Element - The span element is an inline container for text and can be used to group text for styling purposes. However, as span is a generic container to separate pieces of text from a larger body of text, its use should be avoided if a more semantic element is available. <span>...</span>
<a> Anchor Element href="url" target="" The anchor element is used to create hyperlinks in an HTML document. The hyperlinks can point to other webpages, files on the same server, a location on the same page, or any other URL via the hyperlink reference attribute, href. The target attribute on an anchor element specifies where a hyperlink should be opened. A target value of "_blank" will tell the browser to open the hyperlink in a new tab. The anchor element can create hyperlinks to different parts of the same HTML document using the href attribute to point to the desired location with # followed by the id of the element to link to. <a href="www.google.com" target="_blank">Visit this site.</a>

Embedded Media

Tag Name Useful Attributes Description Example Code
<img> image src="url" alt="Text describing the image" HTML image tags embed images in a document. The image tag does not need a closing img tag. <img class="profile-picture" src="./Resources/1.jpg" alt="Photo of Charl de Waal in the Cederberge">
<video> Video src="url" controls Embeds a media player for video playback. The src attribute will contain the URL to the video. Adding the controls attribute will display video controls in the media player. The text between the opening and closing tag is used as a fallback for browsers that do not support the element. <video src="test-video.mp4" controls>Video not supported</video>
<audio> Audio src="url" controls Embed audio into a website. The controls attribute adds audio player controls to the element. <audio src="test.mp3" controls></audio>
<embed> Embed src="url" The embed tag can be used to embed any type of media and does not need a closing tag. <embed src="audio.mp3">
